The online composition feedback tool based on Rietveld is maintained as an open-source project called UCB CodeReview.

A description of the tool and its use in CS 61A at UC Berkeley is described in Teaching Composition Quality at Scale (slides).

We hope to host the tool as a service in the future. In the meantime, Rietveld is hosted as a service already.
